# Тестовое задание для "Специальный технологический центр"

Тестовое задание выполнено с использованием Nuxt 4, с серверным рендерингом (SSR), с TypeScript и SCSS. Проект
представляет собой каталог товаров с адаптивным дизайном и стилизацией по методологии БЭМ.

## Установка

Установка зависимостей:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Сервер разработки

Запустите сервер разработки на `http://localhost:5173`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run 
```
