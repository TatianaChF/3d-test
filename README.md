# Тестовое задание для "Специальный технологический центр"

Тестовое задание выполнено с использованием HTML, CSS, JavaScript и Three.js. Задание
представляет собой группу 3D-объектов и векторный объект, которые реагируют на нажатие кнопки (меняют цвет 
и 3D-объекты меняют свое местоположение).

## Установка

Установка зависимостей:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Сервер разработки

Запустите сервер разработки на `http://localhost:5173`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run 
```
